# Technology Stack Used:
# Frontend :
HTML , CSS , JS
# Backend : 
Node.js , MongoDB , Express

# Packages required :

$ npm install express , body-parser , mongodb

# Start your node server:

$ node index.js

# Start your database server:

$ mongod

# Runs on : localhost:3000
